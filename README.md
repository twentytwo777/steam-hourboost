# Steam-HourBoost
### A script to boost hours in Steam games.

## Where can I find the file to add my games?
#### Follow the path below:
```sh
/steam-hourboost/steam-boost/steam-games.json
```

## How to install?
#### Run the command in the folder with the script:

```sh
npm i
```

## Peculiarities
- Easy to install
- Open source code
